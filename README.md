# ROS-TCP-Connector-Extension

## Introduction
Expand Unity's official [ROS TCP Connector package](https://github.com/Unity-Technologies/ROS-TCP-Connector) and encapsulate functions such as topic subscripting and topic publishing into components.

## How to use
1. Install [ROS TCP Connector package](https://github.com/Unity-Technologies/ROS-TCP-Connector).
2. Install this package from git url "https://github.com/qstx/ROS-TCP-Connector-Extension.git?path=/com.qstx.ros_tcp_ext" in Unity.
3. Add your scripts derived from RosTopicSubscriber class, RosTopicPublisher class or RosServiceClient class.
